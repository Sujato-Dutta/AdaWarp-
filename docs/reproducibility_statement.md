# AdaWarp Reproducibility Statement

## Name

Use **AdaWarp** consistently.  The full expansion is:

**AdaWarp: Adaptive Warped Prototypes for time-series forecasting and classification.**

## Seeds

Main matched-protocol and ablation runs use seeds `42 43 44 45 46`.
Long-term forecasting stress tests default to seed `42` because each neural
baseline sweep is substantially more expensive; increase with
`ADAWARP_LTSF_SEEDS="42 43 44 45 46"` if compute permits.

## TACC Setup

The setup script creates a virtual environment with `--system-site-packages` so
cluster-provided PyTorch/CUDA/NumPy stacks remain visible.  Extra packages are
installed from `requirements-tacc-extra.txt`, which intentionally excludes
`torch`, `numpy`, `tensorflow`, and `jax`.

The original Motion Code matched rerun requires `jax`.  `setup_env.sh` checks
for it but does not install it automatically, because installing JAX can replace
NumPy on some clusters.  Load a TACC-provided JAX module or install it only
after confirming the cluster NumPy stack will not be changed.

Check before launching Motion Code reruns:

```bash
bash scripts/tacc/check_motion_code_env.sh
```

If JAX is missing, original Motion Code will not run. Preferred fix is a TACC
module that provides JAX/JAXLIB. If no module exists and the cluster NumPy stack
is compatible, use `python -m pip install --no-deps jax jaxlib` inside the
AdaWarp venv so pip does not upgrade NumPy/SciPy.

`VPNet` refers to the ICLR'26 variate-patch architecture: patch-level
autoencoding, channelized variate-patch fields, VarTCNBlocks with depthwise 2D
convolutions, pointwise feed-forward mixing, and patch decoding.

```bash
bash scripts/tacc/setup_env.sh
```

To skip all pip installs:

```bash
ADAWARP_INSTALL_EXTRAS=0 bash scripts/tacc/setup_env.sh
```

## Main Runs

Matched Motion Code-style prefix-to-suffix protocol:

```bash
bash scripts/tacc/run_motioncode_protocol.sh
```

This includes the matched original Motion Code forecasting rerun. Do not use
published Motion Code table values in final paper tables.

Long-term forecasting stress test:

```bash
bash scripts/tacc/run_ltsf_main5.sh
```

Core ablations and held-out class-mode forecasting:

```bash
bash scripts/tacc/run_ablation_suite.sh
```

Classification retention reruns:

```bash
bash scripts/tacc/run_classification_retention.sh
```

Aggregate all completed artifacts:

```bash
bash scripts/tacc/aggregate_all.sh
```

## Reporting Policy

Only report values generated by these scripts or by cited published papers.
Do not manually create missing results. Final comparison tables should use
matched TACC reruns; cited published values may be discussed only as background
and must not be mixed into matched-result tables. If a baseline fails because a
package, dataset, or model implementation is unavailable, mark it as not run and
keep the failure log.
